📖 アプリ概要

nainai（ナイナイ）は、不安や悩みを書き出し、光る球体（オーブ）に込めて宇宙へ投げ放つマインドフルネスアプリです。

心のモヤモヤを「書いて捨てる」というセラピー的な行為を、美しいデジタル体験に昇華しました。星空の中で光るオーブをドラッグして飛ばす、その一連の流れが心を軽くしてくれます。

💡 コンセプト・背景

「紙に書いて破って捨てる」というストレス解消法は古くから知られています。nainaiはこのセラピー的行為をデジタルで再現し、さらに美しいビジュアル体験を加えることで、心のケアをより身近で心地よいものにしました。

誰にも見せなくていい。ただ書いて、込めて、手放す――それだけで少し楽になれる。そんな体験を目指しています。

使い方の案内

パーティクルと共に消える

⚡ 主な機能

  • ドラッグ＆スロー操作でオーブを宇宙へ投げる
  • パーティクルエフェクト（オーブが消える際の美しい演出）
  • 星空アニメーション背景
  • 効果音（Web Audio API）による没入感
  • 日本語・英語の切り替え対応
  • PWA対応（ホーム画面に追加可能）
  • オフラインでも動作

🛠 技術スタック

Vanilla JS Canvas API Web Audio API PWA CSS Animations

✦ こだわりポイント

  • ゼロ依存 ― フレームワークを使わず、Vanilla JSのみで構築。軽量で高速
  • プライバシー配慮 ― 入力内容はサーバーに送信されず、どこにも保存されない
  • 物理演算風インタラクション ― ドラッグの速度と角度に応じたオーブの挙動を実現
  • Canvas APIによるリアルタイム描画 ― 星空・オーブ・パーティクルを滑らかに描画
  • Web Audio APIによる効果音 ― 外部ファイル不要のプログラム生成サウンド
